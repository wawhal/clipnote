# ClipNote Icons

This directory contains the extension icons in three sizes:
- icon16.png (16x16) - Toolbar icon
- icon48.png (48x48) - Extension management page
- icon128.png (128x128) - Chrome Web Store

## Creating Icons

You can create icons using any graphics tool. For a quick placeholder:

1. Use a tool like GIMP, Photoshop, or online icon generators
2. Create simple icons with the üìù emoji or "CN" text
3. Save as PNG with transparency
4. Name them: icon16.png, icon48.png, icon128.png

## Temporary Icons

For development, you can use placeholder images. The extension will still work without icons, but Chrome will show a default icon instead.
